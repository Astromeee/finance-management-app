# Changelog

## Version 0.6.0

- Added Upcoming Expenses section with recurring planned expenses, due-date tracking, and mark-as-paid conversion into real expense transactions.
- Reports page redesign: added period filtering, structured financial summaries, category and income breakdowns, spending trends, needs vs wants, account usage, budget performance, upcoming expenses preview, and goals/debts progress.

## Version 0.5.0

- Functional actions: added working income, expense, transfer, and goal flows with separate animated bottom-sheet modals and shared local finance state.
- Added working debt payment flow, local toast feedback, and live account/transaction/budget/goal/debt updates across Dashboard, Accounts, Transactions, Reports, Budgets, and Goals.
- Replaced the generic mock add-transaction flow with action-specific local-state mutations.

## Version 0.4.0

- Accounts redesign: introduced premium wallet-card style account cards, icon-only mobile bottom navigation, professional headers, and manual balance adjustment logic with unexplained income/expense entries.
- Removed casual page greeting text in favor of concise professional page titles and subtitles.
- Replaced the specific fake transfer button with compact account actions.
- Added Unexplained Income and Unexplained Expense mock categories for adjustment transactions.

## Version 0.3.0

- Major UI redesign: replaced gradient-heavy dashboard with a refined premium fintech design system inspired by dark minimal mobile finance apps.
- Introduced graphite surfaces, restrained borders, chartreuse primary accent, and reduced glow/gradient effects.
- Reworked Dashboard hierarchy with balance hero, quick actions, compact summary cards, stronger transaction list, and integrated analytics.
- Refined sidebar, bottom navigation, buttons, forms, progress bars, charts, and transaction styling.
- Updated documentation to describe the new visual system and component rules.

## Version 0.2.0

- Responsive UI polish: improved mobile dashboard, bottom navigation, desktop card layout, add transaction modal, and page-level design consistency.
- Updated greeting and mock personal content for Moeed.
- Reduced mobile card height, added compact dashboard summary cards, and capped chart heights for phones.
- Improved account, transaction, budget, and progress card polish.
- Updated documentation for the refined responsive design.

## Version 0.1.0

- Created React, TypeScript, Tailwind CSS frontend prototype.
- Added Recharts charts, lucide icons, and Framer Motion transitions.
- Built responsive app shell with desktop sidebar and mobile bottom navigation.
- Added Dashboard, Transactions, Accounts, Goals & Debts, Budgets, Reports, and Settings.
- Added Add Transaction modal with income, expense, transfer, goal saving, and debt payment tabs.
- Added mock data, finance TypeScript types, and calculation utilities.
- Added local account transfer demo.
- Added documentation for features, design system, data logic, database plan, user flows, and changelog.
