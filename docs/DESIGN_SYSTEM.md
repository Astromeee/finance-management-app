# Pocket Ledger Design System

The authoritative product design specification is [`../design.md`](../design.md).

The Payday Journey release uses a midnight-plum canvas, persimmon journey/action color, and mint positive state. Home is intentionally limited to one integrated Journey surface, quick actions, one daily prompt, and up to three recent transactions.

Do not reintroduce the retired lime-led palette, decorative glass-card treatment, oversized radii, or dashboard statistic grids. Update `design.md` before changing shared tokens, navigation, typography, component behavior, or accessibility rules.
